# ابدأ من هنا! 🚀

## مرحباً بك في نظام داما للمنح والتأهيل

هذا المشروع يحتوي على **تطبيق React متكامل** لنظام إدارة المنح والتأهيل.

## 📋 الخطوات الأولى (5 دقائق)

### الخطوة 1: التثبيت

```bash
cd frontend
npm install
```

### الخطوة 2: التشغيل

```bash
npm start
```

### الخطوة 3: افتح المتصفح

```
http://localhost:3000
```

**هكذا! التطبيق يعمل الآن** ✅

## 💡 نصائح مهمة

### للتطوير السريع
1. استخدم VS Code
2. ثبت التوسعات المقترحة (في DEVELOPMENT.md)
3. استخدم Redux DevTools
4. استخدم React DevTools

### لحل المشاكل
1. تحقق من الأخطاء في Console (F12)
2. اقرأ SETUP.md في قسم "استكشاف الأخطاء"
3. تأكد من تشغيل Backend على المنفذ الصحيح

1. اقرأ structure المشروع
2. افهم Redux flow
3. تعرف على المكونات الموجودة
