❓ استكشاف الأخطاء:
════════════════════════════════════════════════════════════════════════════════

Docker لا يعمل؟
  → تأكد من تثبيت Docker Desktop
  → تحقق: docker --version

npm install فشل؟
  → احذف node_modules وأعد التثبيت
  → rm -rf node_modules package-lock.json
  → npm install

المنفذ 3000 مستخدم؟
  → غير في .env: PORT=3001

قاعدة البيانات لا تتصل؟
  → تحقق: docker-compose ps
  → أعد التشغيل: docker-compose down && docker-compose up -d
