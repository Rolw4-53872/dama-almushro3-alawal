🚀 البدء السريع (10 دقائق)
════════════════════════════════════════════════════════════════════════════════

1️⃣  بدء قاعدة البيانات:
    $ cd DAMA_Project
    $ docker-compose up -d

2️⃣  تثبيت Backend:
    $ cd backend
    $ npm install

3️⃣  إعداد قاعدة البيانات:
    $ npx prisma migrate dev --name init

4️⃣  شغّل الخادم:
    $ npm run start:dev

5️⃣  افتح في المتصفح:
    http://localhost:3000/api/docs
