📝 ما تم إصلاحه:
════════════════════════════════════════════════════════════════════════════════

✅ تصحيح tsconfig.json:
   - تفعيل experimentalDecorators: true
   - تفعيل emitDecoratorMetadata: true
   - تقليل strictness للسماح بـ decorators

✅ تحديث package.json:
   - إضافة @types/cors
   - إضافة @types/passport-jwt

✅ إصلاح main.ts:
   - تصحيح استيراد helmet و cors
   - استخدام app.enableCors بدلاً من app.use(cors())


🚀 خطوات التشغيل الآن:
════════════════════════════════════════════════════════════════════════════════

# 1️⃣  الذهاب للمجلد
cd C:\Users\RolaA\Desktop\DAMA_Project\backend


# 2️⃣  حذف node_modules والملفات القديمة
